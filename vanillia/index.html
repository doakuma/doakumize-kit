주요 기능

🎨

일관된 디자인 시스템

CSS Variables 기반으로 색상, 타이포그래피, 간격을 통합 관리합니다. 디자인 토큰 한 곳만 수정하면 프로젝트 전체에 일관되게 적용되어 브랜드 아이덴티티를 쉽게 유지할 수 있습니다.

🧩

풍부한 컴포넌트 라이브러리

Button, Input부터 Modal, Tab까지 15개의 완성된 컴포넌트와 24개의 개발 중인 컴포넌트를 제공합니다. 5개 카테고리로 체계적으로 분류되어 필요한 UI를 빠르게 찾을 수 있습니다.

✨

자유로운 커스터마이징

CSS Variables만 수정하면 프로젝트 특성에 맞게 자유롭게 변형할 수 있습니다. 디자인 토큰 구조로 일관성은 유지하면서도 브랜드 컬러와 스타일을 유연하게 조정할 수 있습니다.
