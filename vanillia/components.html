Data 속성 가이드

팝오버를 구현하기 위해 사용되는 data 속성들입니다.

속성명 필수여부 설명 사용 가능한 값
data-popover 필수 팝오버 타입을 지정합니다. simple, details
data-popover-title 선택 팝오버의 제목을 설정합니다. 텍스트 문자열
data-popover-body 선택 팝오버의 본문 내용을 설정합니다. (simple 타입) 텍스트 문자열
data-popover-details 선택 상세 정보를 표시합니다. (details 타입) JSON 배열 [{"label":"...", "value":"..."}]
data-popover-origin 선택 팝오버가 표시될 위치를 지정합니다. right-center(기본), left-center, top-center, bottom-center, top-left, top-right, bottom-left, bottom-right
data-popover-delay 선택 팝오버 표시 지연 시간을 설정합니다. (밀리초) 숫자 (예: 300)
data-popover-variant 선택 팝오버의 크기 변형을 지정합니다. compact, large
data-popover-theme 선택 팝오버의 테마를 지정합니다. dark
