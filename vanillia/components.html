Akumize Design System v1.0

Akumize Component System

왼쪽 메뉴에서 확인하고 싶은 컴포넌트를 선택하면
다양한 변형과 사용 예시를 확인할 수 있습니다.

컴포넌트 시스템 소개

일관성 있고 재사용 가능한 UI 컴포넌트 라이브러리로, 빠르고 효율적인 개발을 지원합니다.

  • 디자인 시스템 기반의 일관된 UI/UX
  • 재사용 가능한 모듈형 컴포넌트
  • 접근성과 사용성을 고려한 설계
  • 실시간 미리보기 및 코드 확인

컴포넌트 종류

다양한 UI 요구사항을 충족하는 15가지 이상의 컴포넌트를 제공합니다.

  • 기본: Button, Input, Checkbox, Radio
  • 선택: Dropdown, Slider, Tab
  • 표시: Table, Chip, File Card, Icon
  • 레이아웃: Modal, Popover, Accordion

스타일 변경 방법

CSS 변수를 활용하여 전체 시스템의 스타일을 한 곳에서 쉽게 관리할 수 있습니다.

  • Color: --primary-*, --gray-*, --success-*
  • Typography: --h1~h5, --body-*, --sub-*
  • Semantic: --text-*, --bg-*, --border-*
/* variables.css에서 변경 */
--primary-500: #6366f1;
--text-primary: var(--gray-900);
