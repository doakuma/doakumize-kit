Component Studio

동일한 디자인 시스템을 여러 프레임워크로 제공합니다

같은 컴포넌트를 반복해서 만드는 수고를 덜고, CSS Variables로 손쉽게 커스터마이징하여 빠르게 프로젝트에 적용할 수 있습니다.

프레임워크를 선택하세요

📦

Vanilla JavaScript

순수 HTML, CSS, JavaScript로 구현된 컴포넌트

  • ✨ 의존성 없음
  • ⚡ 빠른 로딩
  • 🎯 어디서나 사용 가능
  • 🎨 CSS Variables 기반
사용 가능

  • 📦 상태 관리 내장
  • 🌐 생태계 풍부
  • ⚡ Hook 기반

MUI 기반의 Material Design 컴포넌트

  • 🎯 테마 커스터마이징
  • ♿ 접근성 내장
  • 📱 반응형 디자인

✨ 주요 특징

🎨

일관된 디자인 토큰

색상, 타이포그래피, 간격까지 CSS Variables로 중앙 관리됩니다. 하나만 바꾸면 모든 프레임워크에 일관되게 반영됩니다.

🔄

동일한 디자인

Vanilla, React, MUI 모두 동일한 디자인을 제공합니다. 프레임워크가 바뀌어도 사용자 경험은 일관되게 유지됩니다.

📋

복사 & 붙여넣기

코드를 보고 바로 복사해서 사용할 수 있습니다. npm 설치 없이 프로젝트에 즉시 적용 가능합니다.

✨

자유로운 커스터마이징

CSS Variables만 수정하면 브랜드 컬러와 스타일을 원하는 대로 조정할 수 있습니다. 디자인 토큰 구조로 일관성을 유지합니다.

🧩

풍부한 컴포넌트

Button, Input부터 Modal, Tab까지 다양한 컴포넌트를 제공합니다. 체계적으로 분류되어 필요한 UI를 쉽게 찾을 수 있습니다.

⚡

빠른 프로토타이핑

화면 뼈대를 빠르게 만들 수 있습니다. 디자인 시스템이 준비되어 있어 초기 개발 속도가 빨라집니다.
